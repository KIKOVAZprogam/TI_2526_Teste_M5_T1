## Exercícios

1. Explica o que é uma fórmula numa folha de cálculo. Indica dois exemplos de operações que podem ser feitas com fórmulas. (3 valores)

        Resposta: Uma formula é uma equação que podemos introduzir nas folhas de calculo para faciltar os calculos entre celulas.

2. Define “referência relativa” e “referência absoluta”. Dá um exemplo de cada. (3 valores)

        Resposta: Uma referência relativa é uma celula que ajusta-se automaticamente sem necessitar de intervenção humana. Uma referência absoluta é quando uma celula não muda ao copiar ou a arrastar a formula. Referencência relativa: =C2*2      Referência absoluta: =C2*$B$1

4. Para que serve a formatação condicional? Dá dois exemplos práticos. (3 valores)

        Resposta: A formatação condicional permite-nos alterar celulas. Permite-nos alterar coisas como as cores de celulas especificas, sublinhar celulas que atingem certos valores ou adicionar icons ás celulas.

5. Explica o que é um filtro de dados e dá um exemplo prático da sua utilização. (2 valores)

        Resposta: Um filtro de dados é uma ferramenta que permite-nos extrair os dados da tabela, e mostra apenas as linhas de dados que condizem com critérios introduzidos especificos. As linhas que não condizem com os critérios são ocultadas. 

6. Indica a diferença entre “ordenar” e “filtrar” dados. (2 valores)

        Resposta: A função ordenar organiza as linhas da tabela em uma sequência á nossa escolha, enquanto o filtrar oculta dados que não condizem com caracteristicas que escolhemos.

7. Para que serve um gráfico numa folha de cálculo? Indica um tipo de gráfico e uma situação em que possa ser utilizado. (3 valores)

        Resposta: Um gráfico numa folha de cálculo pode ser util para mostrar informação numa forma mais clara. Por exemplo: Numa apresentação, em vez de apenas mostrar a folha de calculos, pode-se mostrar um gráfico de pizza ou de barras para demonstrar as ideias da folha com uma melhor clareza.

8. Escreve a fórmula necessária para calcular a prestação mensal de um empréstimo a 12 meses com taxa anual de 3% sobre 500€. Não é necessário calcular o valor final. (2 valores)

        Resposta: =500*(0,03/12)/(1-(1+0,03/12)^(-12))
